# Brand source drop folder

Place the **provided** logo variants here before running regeneration:

- `logo-2048.png`
- `logo-1024.png`
- `logo-512.png`
- `logo-256.png`
- `logo-128.png`
- `logo-64.png`
- `logo-32.png`
- `logo-16.png`

Optional:

- `logo-48.png` (preferred for extension `48x48`; otherwise `logo-64.png` is copied)
- `favicon.ico` (used as-is when available)
