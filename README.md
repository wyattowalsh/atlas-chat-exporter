# atlas-chat-exporter

Local-first toolkit for exporting Atlas/ChatGPT conversations with shared parsing and rendering logic.

## Branding

All adapters and docs must source branding assets from `assets/brand/` as the canonical source of truth.

See `docs/branding.md` for the branding policy, filename contract, usage mapping, and regeneration flow from provided master variants.

## Brand assets quick reference

- Canonical logo set: `assets/brand/logo-{2048,1024,512,256,128,64,32,16}.png`
- Source drop folder (provided files): `assets/brand/source/`
- Extension/action icons: `apps/extension/assets/icons/`
- Userscript metadata icons: `apps/userscript/assets/icons/`
- Favicons/docs site metadata: `assets/favicon/`
